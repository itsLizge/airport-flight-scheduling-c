# Airport Flight Departure Scheduling Simulation (C)

This project simulates flight departure scheduling at an airport using
queue and priority queue data structures.

## Features
- Flight scheduling based on priority levels
- Simulation of multiple departures
- Analysis of waiting times and queue lengths

## Technologies
- C Programming Language
- Queue & Priority Queue
- Dynamic Memory Management

## How to Run
1. Compile the code:
   gcc main.c -o airport
2. Run:
   ./airport
